# post-it ⬛️

A brutalist note app built with Vue.js.

Inspired by [are.na](https://www.are.na/).

## Project Setup

```
npm install
```

### Compile and Hot-Reload for Development

```
npm run dev
```

### Compile and Minify for Production

```
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```
npm run lint
```
